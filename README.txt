Description : application web permettant aux utilisateurs de gérer une liste de tâches.
 
1. Interface Utilisateur :
	--> Afficher une interface utilisateur interactive pour ajouter, modifier et supprimer des tâches.

2. Ajout et Suppression de Tâches :
	--> Permettre d'ajouter de nouvelles tâches et de les supprimer une fois accomplies.

3. Marquage des Tâches Accomplies :
	--> Permettre aux utilisateurs de marquer les tâches comme accomplies.

4. Filtrage des Tâches : 
	--> Ajouter des filtres pour afficher les tâches accomplies ou non accomplies.

5. Sauvegarde Locale des Données :
	--> Assurer que les tâches restent même après le rafraîchissement de la page.
